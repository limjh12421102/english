Tip: 필수 기본 문법(숲) 를! 먼저 알고 세부적인 것(나무) 나올때마자 정리해 나가자~!


* waters : 바다



Tip: 기호(수단)에 집착하지 말자.



* 서수(순서를 나타내는 수) 앞에 the(정관사) 를 ! 쓴다.


* story ( ~ 짜리 층 )  ,  floor ( 구체적인 층 )



* 유일한 것 앞에 the 를 ! 붙힌다.


* the  +  ⓐ 

    1) ⓐ 한 사람들          =>      복수 취급!

        ex) the young == young people

    2) ⓐ 한 것(추상명사)    =>      단수 취급!

        ex) the beautiful == 아름다움


* Japan

    Japanese ⓐ 일본의 ⓝ 일본어 == the Japanese language

    the Japanese == Japanese people




* the + 복수        =>      전체!


    ex) the Smiths == 스미스 가족들






* leave for : ~(으)로 향해 떠나다







* 대명사쓰는 목적 : For 중복회피




* 비인칭주어가 있듯이 Vt의 목적어 는 반드시 필요한데 마땅히 쓸 만 한 것이 없으면 "it" 를! 쓴다.

    ex) I couldnt' help it
    ex) Let's call it a day. 





* ~ 를 졸업하다 : graduate from 

    - 이때 graduate 은 자동사





* 전치사 + 명사 == 구 ( 형용사구 oR 부사구 )


    - 명사는 전치사의 대상이 된다

    - 명사 == 전치사의 목적어




* considerate = thoughtful : 사려깊은, 이해심 많은



* considerable : 고려할 만한; 상당한






* 재귀대명사의 관용적 표현 (암기)


    - introduce oneself     : 자기소기하다
    - say[talk] to oneself  : 혼잣말하다
    - make oneself at home  : 느긋하게편히 쉬다
    - by oneself            : 홀로; 혼자힘으로
    - help oneself to       : (음식들을) 자유로이 먹다





* 비교 & 대조의 두대상은 동일 조건이어야!

    your idea 와 비교할 수 있는 것은 me 가 아니라 my idea 이다.





* bark ⓥ 짖다 ⓝ 나무껍질



* 명사's 

    1) 소유격

    2) 소유대명사




* 단위는 셀 수 (o) 



Tip: 나올떄마다 복습한다면! 언젠가는 외워지겠지.







* one (대명사)

    - 앞에서 나온 셀 수 있는 명사를! 대신 받는다. ( 형용사의 수식을 받고있는 상태임. )

        ex) The green dress => favorite one

    - 복수일 땐 ones





* 영어에서 명사가 동사자리에 오면 동사로써 쓰이는 경우가 多

    ex) water the garden ( ~에 물을 주다 )

    ex) button the long coat ( ~의 단추를 잠구다 )





* -ity 로 끝나는 단어는 바로 앞에 강세가 온다.

    ex) faci`ltiy  [퍼`씰 러리]
